# Useful commands


## Run tests:

```shell
./gradlew :unde-android-library:testDebugUnitTest --tests "com.unde.library.UndeLibraryTest"
```


## Build debug version:

```shell
./gradlew :unde-android-library:assembleDebug
```

## Build release version:

```shell
./gradlew :unde-android-library:assembleRelease
```
